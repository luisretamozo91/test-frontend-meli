# test-frontend-meli
Test práctico para aspirantes al área de front-end de Mercado Libre

Cliente:
- HTML
- React
- Sass

Servidor:
- Node
- Express

## instalacion
- Clonar o descargar este repositorio
- Instalar nodemon: `npm i nodemon -g`
- 
```
yarn or npm install
cd client
yarn or npm install
```

## iniciar servidor y cliente
``
yarn dev
``
